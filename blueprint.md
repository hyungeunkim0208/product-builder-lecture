# Lotto Number Drawing Site

## Overview

This is a simple web application that generates random lotto numbers.

## Features

*   Generate 6 unique random numbers between 1 and 45.
*   Display the generated numbers.
*   A button to trigger the number generation.

## Plan

*   **index.html**: Create the basic structure of the web page, including a title, a button, and a container to show the numbers.
*   **style.css**: Style the page to be visually appealing, with a "lotto" theme.
*   **main.js**: Implement the logic for generating random numbers and updating the display.
